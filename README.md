# 🎬 Netflux — Premium Netflix Clone (Static Frontend)

### 🚀 Live Demo: [netflux-piyush.vercel.app](https://netflux-piyush.vercel.app/)

A high-fidelity, pixel-perfect frontend clone of **Netflix** built entirely using **Pure HTML, CSS, and Vanilla JavaScript** — without any external frameworks, libraries, or APIs. 

This project features premium design aesthetics, interactive components, responsive web layouts, and custom AI-generated movie poster images to deliver a truly cinematic user experience.

---

## ✨ Features Implemented

*   **🎬 Cinematic Hero Banner:** A full-viewport spotlight spotlighting "Stranger Dimensions" with a slow Ken Burns zoom effect and gradient overlays for text contrast.
*   **⚡ Skeleton Shimmer Loading:** Smooth pre-loaders shimmer for `800ms` when the page loads, simulating data fetching before transitioning into the loaded interface.
*   **🔍 Live Instant Search:** Click-to-expand search bar in the navbar that dynamically filters titles from all category rows on typing.
*   **📱 Scroll-Adaptive Navbar:** Elegant transparent navbar at the top that smoothly transitions to solid black with a subtle shadow when scrolling.
*   **↔️ Responsive Scrollable Rows:** Category rows (Trending, Originals, Top-rated, etc.) with touch scroll support and clean hover-reveal scroll arrows.
*   **🎬 Dynamic Detail Modals:** Click on any movie card to trigger a premium, blurred backdrop overlay containing a full synopsis, cast lists, matches, and tags.
*   **✨ Interactive Micro-animations:** Micro-interactions on buttons, scale-up transformations on hover, and active states.
*   **📱 Fully Responsive Design:** Media queries optimized for Desktop, Tablet, and Mobile viewport sizes.

## 🎨 Tech Stack & Design System

*   **HTML5 & CSS3:** Semantic structure & styling.
*   **Vanilla JS (ES6+):** Responsive events, DOM manipulation, custom IntersectionObservers.
*   **Typography:** Google Font `Inter` (weights: 400, 500, 600, 700).
*   **Colors:**
    *   Primary: `#e50914` (Netflix Red)
    *   Background: `#141414` (Netflix Dark)
    *   Surface: `#181818` (Netflix Cards)
    *   Match Text: `#46d369` (Netflix Green)

## 🚀 How to Run Locally

Since this is a fully static project, there are no dependencies to install:
1. Clone this repository or download the ZIP.
2. Open the directory `netflix-clone/`.
3. Double-click the **`index.html`** file to view the application instantly in your browser.
